
这个小工具的作用:  
在一个项目内做题.  一道题一个rs文件.  根据做题平台划分. 
利用vs code的测试按钮运行编写的答案.   


## 需要准备: 


- 创建一个rust项目. 
- 创建下列目录, 每个目录中创建`mod.rs`文件. 
- `lib.rs` 需要包含下列目录:  `leetcode` `luogu` `atcoder`



## 使用: 

在做题项目的目录下执行`zuoti_initer 平台简写  题号`  

- leetcode 简写: lc
- luogu 简写: luogu
- atcoder 简写: at
